# Полная инструкция по развёртыванию системы

Принцип: **всё что можно автоматизировать — автоматизировано.** Руками — только то, что требует аккаунтов и кликов в браузере.

---

## ВАЖНО: Транскрипция голосовых (Буквица)

Система транскрибирует голосовые через Telegram-бот [Буквица](https://bykvitsa.ru/) ([@BukvitsaAI_bot](https://t.me/BukvitsaAI_bot)). Бот отправляет голосовое в Буквицу через Telethon (клиентский API Telegram), получает текст обратно.

**Это означает:**
- Для транскрипции нужен Telethon (API_ID + API_HASH + авторизация по номеру телефона)
- Буквица — внешний платный сервис (https://bykvitsa.ru/), не контролируется нами (8 т.р./год подписка)
- Если Буквица недоступна или нужна другая система — в bot.py нужно заменить функцию `transcribe_via_bukvitsa()` на альтернативу

**Варианты транскрипции:**
1. **Буквица (текущий)** — платно, работает через Telethon, качество отличное для русского
2. **OpenAI Whisper API** — платно (~$0.006/мин), нужен OPENAI_API_KEY, заменить одну функцию
3. **Локальный Whisper** — бесплатно, нужен GPU или терпение, заменить одну функцию

Решить до начала настройки. Если Буквица — нужен Telethon (шаг 3). Если другое — Telethon не нужен, но нужен соответствующий API-ключ.

## Шаг 0. Скачать проект

4. Перейти в папку, где хочешь хранить проект:

```bash
cd ~/Documents
```

5. Склонировать:
```bash
git clone https://github.com/bullet382334/AISKRA.git
```

## Шаг 0. Что потребуется создать руками

| Что | Где создать | Что получишь |
|-----|-----------|-------------|
| Telegram-бот | @BotFather в Telegram | BOT_TOKEN |
| Telegram-группа | Telegram | GROUP_CHAT_ID (скрипт получит сам) |
| Telethon API | my.telegram.org | API_ID, API_HASH |
| Notion интеграция | notion.so/my-integrations | NOTION_API_TOKEN |
| Notion страница | Notion | NOTION_ROOT_PAGE |
| Unsplash (опционально) | unsplash.com/developers | UNSPLASH_ACCESS_KEY |

---

## Шаг 1. Создать Telegram-бота

1. Открыть Telegram, найти **@BotFather**, нажать **Start**
2. `/newbot` → задать имя и username
3. Записать **BOT_TOKEN**

### Настроить бота
4. `/setprivacy` → выбрать бота → **Disable** (обязательно!)
5. `/setcommands` → выбрать бота → отправить ОДНИМ сообщением:
```
push - Обработать буфер транскрипций через Claude
status - Статус буфера и системы
buffer - Показать содержимое буфера
plan - Показать текущий план задач
research - Исследование по теме
do - Выполнить конкретную задачу
sync - Синхронизировать карточки в Notion
catchup - Проверить пропущенные сообщения группы
clear - Очистить буфер транскрипций
```

---

## Шаг 2. Создать Telegram-группу и добавить бота

1. Новая группа → добавить заказчика и/или себя
2. Добавить бота по username → **Promote to Admin** (Send Messages, Delete Messages)

**GROUP_CHAT_ID и ADMIN_CHAT_ID получит скрипт автоматически (шаг 7).**
Для этого: написать боту лично + написать в группу.

---

## Шаг 3. Получить Telethon API (если используешь Буквицу)

1. **https://my.telegram.org/** → ввести номер → код → **API development tools**
2. Создать приложение → записать **TELETHON_API_ID** и **TELETHON_API_HASH**

---

## Шаг 4. Создать Notion workspace и интеграцию

1. **https://www.notion.so/** → войти или зарегистрироваться
2. **https://www.notion.so/my-integrations** → **+ New integration** → записать **NOTION_API_TOKEN**
3. Создать корневую страницу → подключить интеграцию → записать **NOTION_ROOT_PAGE** (последние 32 символа URL)

---

## Шаг 5. Unsplash API (опционально)

Обложки для карточек в Notion.

1. **https://unsplash.com/developers** → **New Application** → записать **UNSPLASH_ACCESS_KEY**

---

## Шаг 6. Записать все токены

Создать `secrets.txt` (в .gitignore):
```
BOT_TOKEN=токен_из_шага_1
TELETHON_API_ID=число_из_шага_3
TELETHON_API_HASH=строка_из_шага_3
NOTION_API_TOKEN=ntn_из_шага_4
NOTION_ROOT_PAGE=id_страницы_из_шага_4
UNSPLASH_ACCESS_KEY=ключ_из_шага_5
PERSON_TG_USERNAME=username_заказчика_без_собаки
MAIN_PROJECT_KEYWORD=ключевое_слово_проекта
```

## Шаг 7. Запустить setup.py

```bash
python setup.py
```

Автоматически: проверяет Python, устанавливает зависимости, читает secrets.txt, создаёт .env файлы, получает chat ID, проверяет токены, настраивает фоновый запуск.

Бот вызывает Claude через `npx claude`. **Требуется подписка Anthropic** (Claude Max или Pro).
