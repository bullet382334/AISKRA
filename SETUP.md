Принцип: **всё что можно автоматизировать — автоматизировано.** Руками — только то, что требует аккаунтов и кликов в браузере.

## ВАЖНО: Транскрипция голосовых (Буквица)

Система транскрибирует голосовые через Telegram-бот [Буквица](https://bykvitsa.ru/) ([@BukvitsaAI_bot](https://t.me/BukvitsaAI_bot)). Бот отправляет голосовое в Буквицу через Telethon (клиентский API Telegram), получает текст обратно.

**Это означает:**
- Для транскрипции нужен Telethon (API_ID + API_HASH + авторизация по номеру телефона)
- Буквица — внешний платный сервис (https://bykvitsa.ru/), не контролируется нами (8 т.р./год подписка)
- Если Буквица недоступна или нужна другая система — в bot.py нужно заменить функцию `transcribe_via_bukvitsa()` на альтернативу

**Варианты транскрипции:**
1. **Буквица (текущий)** — платно, работает через Telethon, качество отличное для русского
2. **OpenAI Whisper API** — платно (~$0.006/мин), нужен OPENAI_API_KEY, заменить одну функцию
3. **Локальный Whisper** — бесплатно, нужен GPU или терпение, заменить одну функцию

Решить до начала настройки. Если Буквица — нужен Telethon (шаг 4). Если другое — Telethon не нужен, но нужен соответствующий API-ключ.

## Шаг 1. Установить VS Code + Claude Code (рекомендуется)

VS Code — бесплатный редактор кода. Claude Code — расширение-помощник: видит все файлы проекта, запускает команды, исправляет ошибки и ведёт шаг за шагом.

> 💡 **Зачем:** ниже будут шаги со скриптами, токенами, запуском. Если что-то пойдёт не так — Claude сразу увидит ошибку и подскажет решение. Без него придётся разбираться самому.

**Установка VS Code:**
1. Скачать: **https://code.visualstudio.com/** → нажать **Download** → установить с настройками по умолчанию

**Установка расширения Claude Code:**
1. Открыть VS Code → в боковой панели слева нажать значок **Extensions** (квадратики, или `Ctrl+Shift+X`)
2. В поиске набрать **Claude Code** → найти расширение от **Anthropic** → нажать **Install**
3. После установки в боковой панели появится иконка Claude → нажать на неё
4. Авторизоваться через аккаунт Anthropic (нужна подписка Claude Max или Pro)

**Открыть проект:**
1. В VS Code: **File → Open Folder** (Файл → Открыть папку) → выбрать папку AISKRA (ту, что скачали на шаге 0)
2. Нажать на иконку Claude в боковой панели — откроется чат

**Дальше можно идти двумя путями:**
- **С Claude:** шаги 2–6 (создание аккаунтов, сбор токенов) — делаешь в браузере сам. Начиная с шага 7 — пиши Claude, он поможет:
  - `Я собрал все токены, помоги создать secrets.txt и запустить setup.py`
  - `Запусти setup.py и помоги, если будут ошибки`
  - `Запусти бота и проверь, что он работает`
  - `Что-то не работает, помоги разобраться`
- **Без Claude:** продолжай по инструкции ниже, всё расписано пошагово

> Если не хочешь ставить VS Code — можно работать без него. Все шаги ниже самодостаточны.

## Шаг 3. Создать Telegram-группу и добавить бота

1. Открыть Telegram → значок карандаша (написать сообщение) → **Создать группу** (New Group)
2. Название — любое понятное, например «Идеи — Имя» или «AISKRA — Имя»
3. Добавить участников: себя и/или заказчика. Нажать **Создать** (Create)
4. Добавить бота в группу:
   - Открыть группу → нажать на название группы вверху → **Участники** (Members) → **Добавить участника** (Add Members)
   - Найти бота по username из шага 2 (например @my_iskra_bot) → добавить
5. Сделать бота администратором:
   - Нажать на название группы вверху → **Участники** (Members)
   - Найти бота в списке → нажать на него → **Назначить администратором** (Promote to Admin)
   - Включить права: **Отправка сообщений** (Send Messages), **Удаление сообщений** (Delete Messages)
   - Остальные права можно не включать → **Сохранить** (Save)

6. **Сразу после создания группы** — отправить два сообщения (любой текст, например «привет»):
   - Боту лично (открыть бота → написать) — чтобы скрипт узнал ADMIN_CHAT_ID
   - В группу — чтобы скрипт узнал GROUP_CHAT_ID

> Это нужно для шага 8 (setup.py). Скрипт найдёт ID автоматически по этим сообщениям.

## Шаг 4. Получить Telethon API (если используешь Буквицу)

1. Открыть в браузере **https://my.telegram.org/**
2. Ввести номер телефона в международном формате (например +7 999 123 45 67) → нажать **Next**
3. В Telegram придёт код подтверждения (сообщение от Telegram, не SMS) → ввести его на сайте
4. После входа нажать **API development tools**
5. Заполнить форму:
   - **App title** — любое название, например «AISKRA»
   - **Short name** — короткое имя латиницей, например «aiskra»
   - **Platform** — выбрать **Desktop**
   - **Description** — можно оставить пустым
   - Нажать **Create application**
6. На экране появятся **App api_id** (число) и **App api_hash** (длинная строка из букв и цифр)
7. Записать оба значения — это **TELETHON_API_ID** и **TELETHON_API_HASH**

> ⚠️ Эти данные создаются один раз и не меняются. Не делись ими — это доступ к твоему аккаунту Telegram.

## Шаг 5. Создать Notion workspace и интеграцию

### 5а. Аккаунт Notion
1. Открыть **https://www.notion.so/** → нажать **Get Notion free** (или **Войти**, если уже есть аккаунт)
2. Зарегистрироваться через email или Google-аккаунт
3. При первом входе Notion создаст workspace автоматически — это нормально

### 5б. Создать интеграцию (получить API-токен)
1. Открыть **https://www.notion.so/my-integrations** (нужно быть залогиненным)
2. Нажать **+ New integration** (кнопка в правом верхнем углу)
3. Заполнить:
   - **Name** — любое, например «AISKRA»
   - **Associated workspace** — выбрать свой workspace (обычно уже выбран)
   - **Type** — оставить **Internal**
4. Нажать **Save** (Сохранить)
5. На следующем экране — нажать **Show** рядом с **Internal Integration Secret** → скопировать токен
6. Это **NOTION_API_TOKEN** — начинается с `ntn_` (записать)

### 5в. Создать корневую страницу и подключить интеграцию
1. Вернуться в Notion → в боковой панели слева нажать **+ Add a page** (+ Добавить страницу)
2. Назвать страницу, например «AISKRA» или «Карта идей»
3. Подключить интеграцию к странице:
   - На открытой странице нажать **···** (три точки, правый верхний угол) → **Connect to** (Подключения)
   - Найти свою интеграцию (AISKRA) → нажать **Confirm** (Подтвердить)
4. Скопировать ID страницы из URL:
   - Посмотреть адресную строку браузера, URL выглядит так: `https://www.notion.so/Название-**xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx**`
   - Последние 32 символа (после последнего дефиса) — это **NOTION_ROOT_PAGE** (записать)

## Шаг 7. Записать все токены

Все собранные токены нужно записать в один файл. Этот файл НЕ попадёт в git (он в .gitignore).

1. Открыть папку проекта AISKRA
2. Создать текстовый файл с именем `secrets.txt` (не `secrets.txt.txt` — убедись, что расширение именно `.txt`)
   - **Windows**: правый клик в папке → Создать → Текстовый документ → переименовать в `secrets.txt`
   - **Mac**: открыть TextEdit → Формат → Сделать простым текстом → сохранить как `secrets.txt` в папку проекта
3. Вставить в файл строки ниже, заменив значения на свои (без кавычек, без пробелов вокруг `=`):

```
BOT_TOKEN=токен_из_шага_2
TELETHON_API_ID=число_из_шага_4
TELETHON_API_HASH=строка_из_шага_4
NOTION_API_TOKEN=ntn_из_шага_5
NOTION_ROOT_PAGE=id_страницы_из_шага_5
UNSPLASH_ACCESS_KEY=ключ_из_шага_6
PERSON_TG_USERNAME=username_заказчика_без_собаки
MAIN_PROJECT_KEYWORD=ключевое_слово_проекта
```

4. **PERSON_TG_USERNAME** — username заказчика в Telegram без символа @. Например, если у него `@ivan_petrov`, пишем `ivan_petrov`
5. **MAIN_PROJECT_KEYWORD** — ключевое слово проекта (одним словом, латиницей), используется для именования файлов
6. Сохранить файл

## Шаг 9. Установка Claude Code CLI

Бот обрабатывает сообщения, вызывая Claude через терминальную команду `npx claude`. Это **не то же самое**, что расширение в VS Code — расширение помогает тебе, а CLI нужен самому боту для работы.

1. Открыть терминал (или терминал внутри VS Code: **Terminal → New Terminal**)
2. Установить:
```bash
npm install -g @anthropic-ai/claude-code
```
3. Авторизоваться:
```bash
claude
```
Claude откроет браузер для входа через Anthropic-аккаунт. После авторизации можно закрыть — CLI готов.
