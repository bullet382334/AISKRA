# Workflow адаптивного интервью

> Процесс интервью — не фиксированный опросник, а живая система, которая подстраивается под ответы заказчика.

---

## Принцип

```
Ответ заказчика → Анализ → Генерация следующих вопросов → Ответ → ...
```

Базовые вопросы (в `bazovye-voprosy.md`) — это стартовая точка. После первых ответов каждая следующая серия генерируется на основе анализа собранного.

---

## Workflow после каждого ответа

### Шаг 1: Фиксация ответа
- Сохранить транскрипцию в `intervyu/otvety/ГГГГ-ММ-ДД-seriya-N.md`
- Указать: на какой вопрос отвечала, контекст, длительность.

### Шаг 2: Анализ ответа
Создать файл `intervyu/analiz/ГГГГ-ММ-ДД-analiz-serii-N.md` по шаблону `shablon-sleduyushchej-serii.md`.

Анализировать по 7 осям:

| Ось | Вопрос | Что записать |
|-----|--------|-------------|
| **Раскрытые темы** | Какие темы заказчик раскрыл полно? | Список тем с пометкой «раскрыта» |
| **Повторения** | Какие мысли уже звучали раньше? | Мотив + сколько раз + в каком контексте |
| **Энергия** | Где голос оживлён, где горит? | Конкретные фразы и темы с высокой энергией |
| **Неясности** | Что осталось размытым, неконкретным? | Конкретные места, где нужно уточнение |
| **Противоречия** | Где она говорит одно, а раньше — другое? | Два противоречащих утверждения |
| **Пробелы** | О чём вообще не говорила? | Темы, которых не касалась |
| **Перспективные нити** | Что выглядит живым и стоит углубить? | Темы, в которых чувствуется потенциал |

### Шаг 3: Обновление базы знаний
- Обновить `karta-idej/digest.md` новыми знаниями.
- Создать карточки для новых сущностей.
- Обновить `kontekst/profil-persona.md` (если есть новое).
- Обновить главный проект (если есть новое).

### Шаг 4: Генерация следующих вопросов

На основе анализа сформулировать **5 вопросов** по правилам:

**Выбрать стратегию для каждого вопроса:**

| Стратегия | Когда | Пример |
|-----------|-------|--------|
| Углубление | Тема живая, есть энергия, но поверхностно | «Ты говорила про X — расскажи подробнее, как ты это видишь?» |
| Прояснение | Есть неясность или противоречие | «Ты упомянула X и Y — как они связаны для тебя?» |
| Расширение | Важная тема не затронута | «А что ты думаешь про Z?» |
| Конкретизация | Идея есть, но нет деталей | «Если бы ты начала с одного шага — что бы это было?» |
| Проверка мотива | Мысль повторяется, нужно понять глубину | «Ты уже несколько раз говорила про X — почему это так важно?» |

**Правила формулировки:**
- Короткие (1–2 предложения).
- Открытые (не да/нет).
- На языке заказчика, не на языке аналитики.
- Не больше 5 за раз.
- Не повторять уже раскрытое (если заказчик уже ответил полно — двигаться дальше).
- Каждый вопрос — с обоснованием «почему именно этот».

### Шаг 5: Оформление и отчёт
- Записать вопросы в `intervyu/analiz/ГГГГ-ММ-ДД-analiz-serii-N.md`.
- Сообщить пользователю: краткий итог + предложенные вопросы + обоснования.

## Темы для отслеживания

Эти темы нужно постепенно раскрыть через интервью. Не все сразу — по мере естественного хода разговора.

### О заказчике
- [ ] Образ идеального будущего
- [ ] Что зажигает / что забирает энергию
- [ ] Текущие занятия и доход
- [ ] Сильные стороны (как сам их видит)
- [ ] Желаемая роль в проекте / жизни
- [ ] Отношение к рутине, бюрократии, финансам
- [ ] Что НЕ хочет делать

### О главном проекте
- [ ] Почему именно этот проект / место
- [ ] Образ места (что там будет)
- [ ] Для кого (семья, гости, туристы)
- [ ] Готовность к жизни там (сезонно / постоянно)
- [ ] Форматы деятельности и заработка
- [ ] Страхи и тревоги

### О семье и ролях
- [ ] Кого видит рядом в проекте
- [ ] Какую роль хочет для себя
- [ ] Как представляет участие семьи
- [ ] Финансовые ожидания

## Когда переходить к следующей стратегии

| Сигнал | Действие |
|--------|----------|
| Заказчик отвечает коротко, без энергии | Сменить тему, не давить |
| Один и тот же мотив 3+ раз | Перестать спрашивать, зафиксировать как устойчивый |
| Противоречие | Мягко прояснить: «Ты говорила X, а сейчас Y — как это сочетается?» |
| Все базовые темы раскрыты | Перейти к конкретизации и проверке |
| Заказчик сам поднимает новую тему | Следовать за ним, не возвращать к плану |
