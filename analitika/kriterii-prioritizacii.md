# Критерии приоритизации идей

---

## Шкала оценки (1–5 по каждому критерию)

| Критерий | Вопрос | 1 (низкий) | 5 (высокий) |
|----------|--------|------------|-------------|
| **Энергия заказчика** | Насколько это зажигает? | Упомянул раз | Говорит постоянно, горит |
| **Повторяемость** | Сколько раз возвращалась к этой мысли? | Один раз | 5+ раз в разных контекстах |
| **Реалистичность** | Можно ли это сделать с текущими ресурсами? | Фантазия | Конкретный план |
| **Ресурсы** | Есть ли деньги, время, люди? | Ничего нет | Всё есть |
| **Связь с главным проектом** | Это усиливает главный проект? | Не связано | Ключевая часть |
| **Семейное согласие** | Семья поддерживает? | Против | Единогласно за |
| **Временные рамки** | Есть ли окно возможности? | Когда-нибудь | Нужно сейчас, иначе уйдёт |

## Как считать

Просто сложить баллы. Не нужна точность — нужно сравнение.

| Сумма | Приоритет |
|------:|-----------|
| 28–35 | Высокий — действовать |
| 20–27 | Средний — проработать |
| 12–19 | Низкий — отложить |
| 7–11 | Минимальный — записать и забыть пока |

## Пример

| Критерий | Проект А | Проект Б |
|----------|:-------------------:|:------------------:|
| Энергия | 5 | 3 |
| Повторяемость | 4 | 1 |
| Реалистичность | 3 | 4 |
| Ресурсы | 2 | 3 |
| Связь с главным проектом | 5 | 1 |
| Семейное согласие | 3 | 3 |
| Временные рамки | 3 | 1 |
| **Итого** | **25** | **16** |

---

> Применять, когда нужно выбрать между несколькими направлениями.
> Не использовать для убийства идей — только для приоритизации.
