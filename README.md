# Система сбора и развития идей

---

## Зачем нужна эта система

Система для человека с идеями, которому нужна структура. Идеи приходят быстро, в разных форматах, часто голосом. Без системы они теряются.

**Система** — это:
- место, где ни одна идея не теряется;
- инструмент, чтобы отличить вспышку от устойчивой мечты;
- площадка для обсуждения замыслов, ресурсов и рисков;
- рабочая система для перехода от «хочу» к «делаем».

---

## Как работает

### Сбор → Обработка → Анализ → Действие

```
Голосовое / сообщение в Telegram
        ↓
   inbox/ (сырой материал)
        ↓
  AI обработка: выделение мыслей, создание карточек
        ↓
  karta-idej/ (идеи, гипотезы, проекты, риски...)
        ↓
  analitika/ (критическая проверка — отдельно от сбора!)
        ↓
  Решение → Следующие шаги → Сводка
        ↓
  Адаптивные вопросы → следующее интервью
```

## С чего начать

1. Настроить систему по [SETUP.md](SETUP.md)
2. Прочитать [intervyu/bazovye-voprosy.md](intervyu/bazovye-voprosy.md) и задать первые 3 вопроса.
3. Когда заказчик ответит — `/push` в боте.
4. AI обработает, предложит следующие вопросы.
5. Повторять цикл.
