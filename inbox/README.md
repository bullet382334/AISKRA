# Inbox — входящие материалы

Сюда попадает всё, что ещё не разобрано:
- транскрипты голосовых;
- заметки из разговоров;
- ссылки;
- пересланные сообщения.

## Правила

1. **Всё входящее — сюда.** Не пытаться сразу классифицировать.
2. **Имя файла:** `ГГГГ-ММ-ДД-краткая-тема.md`
3. **Минимум в файле:** дата, источник, текст как есть.
4. **Разбор:** по workflow из `pravila/workflow.md`.
5. **После разбора:** файл не удаляется, в шапку добавляется `разобрано: да`.
