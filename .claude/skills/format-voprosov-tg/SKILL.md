Формат сообщений для Telegram — строго так:

## Жирный текст
IMPORTANT: При отправке через Telegram Bot API использовать `parse_mode: "HTML"` и теги `<b>текст</b>`.
НЕ использовать Markdown (**) — Telegram его не рендерит с parse_mode Markdown надёжно.
Выделять жирным **важное** в каждом сообщении — то, на что получатель должен обратить внимание.

При показе текста пользователю в чате — использовать `**текст**` (markdown), чтобы было читаемо.

## Вопросы для заказчика

```
[эмодзи] [номер]. **[Короткий заголовок]**

[Текст вопроса]
```

Пример:
```
✨ 4. **Ресурсы**

Из всех направлений, которые ты пробовал(а) — где теряешь счёт времени? Где забываешь, что это «работа»?
```

Каждый вопрос — отдельным сообщением. Эмодзи подбирать по теме.
Всегда выдавать в этом формате, готовым к копированию в Telegram.
