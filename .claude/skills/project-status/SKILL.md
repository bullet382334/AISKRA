## Что показать пользователю

Сформируй краткую таблицу:

| Метрика | Значение |
|---------|----------|
| Карточек идей | [число] |
| Серий интервью | [число] отправлено, [число] обработано |
| Неразобрано в inbox | [число] |
| Последний коммит | [описание] |
| digest.md | [число] / 100 строк |
| _sostoyaniye.md | [число] / 40 строк |

Плюс:
- Что ждём (из _sostoyaniye.md)
- Следующий шаг
- Проблемы (если есть: превышены лимиты, есть неразобранное)
