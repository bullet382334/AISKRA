## Формат TG-сообщений по исследованиям для группы заказчика

IMPORTANT: При создании исследования в `realizaciya/` — ОБЯЗАТЕЛЬНО создавать файл `bot/msg-{slug}.txt` с сообщениями для Telegram-группы заказчика.

### Цепочка
1. Claude создаёт `realizaciya/{slug}.md` (исследование)
2. Claude создаёт `bot/msg-{slug}.txt` (сообщения для группы)
3. Бот запускает Notion sync
4. После sync бот автоматически отправляет сообщения в группу заказчика

### Формат файла `bot/msg-{slug}.txt`

- Блоки разделены `---`
- Каждый блок = одно сообщение
- HTML parse_mode: `<b>жирный</b>` (НЕ Markdown **)
- Эмодзи — по теме каждого блока

### Структура сообщений

**Блок 1 — Заголовок + суть:**
```
[эмодзи] <b>[Название исследования]</b>

[1-2 строки: что это и зачем. Контекст для заказчика.]
```

**Блоки 2-N — Чеклист «Что делать», разбитый по темам:**
Извлечь из секции `## Что делать` исследования. Сгруппировать по темам:
```
[эмодзи] <b>[N. Тема группы]</b>

[действие 1]
[действие 2]
...
```

Правила группировки:
- Юридическое → ⚖️
- Финансы / ипотека / расчёты → 💰
- Осмотр / проверка на месте → 🔍
- Документы → 📄
- Коммуникации / инфраструктура → 🔌
- Уже проверено / ОК → ✅
- Риски / внимание → ⚠️
- Итог / срочное → 🔑

**Последний блок — самое срочное:**
```
🔑 <b>Самое срочное:</b> [что делать прямо сейчас — 1-2 предложения]
```

### Акценты

Выделять `<b>жирным</b>`:
- Названия действий (что сделать)
- Ключевые цифры
- Статусы (✓ проверено, ⚠️ внимание)
- Имена, даты, суммы

### Пример

```
🏠 <b>Объект — что нужно сделать до покупки</b>

По результатам анализа — вот полный чеклист.
---
⚖️ <b>1. Юридическое (критично)</b>

📋 <b>Свежая выписка ЕГРН</b> — для сделки нужны не старше 30 дней
🗺 <b>Проверить ЗОУИТ</b> через публичную кадастровую карту
👨‍⚖️ <b>Консультация юриста</b> — по ЦЭЗ, ФЗ-482
---
✅ <b>6. Что уже проверено — ОК</b>

• Обременений нет (ЕГРН)
• ВРИ = ИЖС (ЕГРН)
• Единственный собственник
---
🔑 <b>Самое срочное:</b> юрист по ЦЭЗ/ФЗ-482 + свежая выписка ЕГРН.
```

### Что НЕ включать
- Детали исследования (цифры рынка, сравнения, источники) — для этого есть Notion
- Длинные цитаты из файла
- Более 10 сообщений на одно исследование
