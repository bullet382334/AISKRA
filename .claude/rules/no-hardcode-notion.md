## Запрет хардкода в Notion sync

IMPORTANT: НИКОГДА не добавлять файлы/карточки вручную в `update_notion.py`.

- `realizaciya/` — файлы подхватываются автоматически функцией `discover_realizaciya_cards()`
- Новый файл в папке = автоматическая карточка в Notion при следующем sync
- Заголовок и статус читаются из первых строк .md файла (формат: `# Заголовок` + `> Статус: ...`)
- Дочерние файлы (kp-tekst-*.md) исключены через `_CHILD_FILES`

Если нужно добавить новую папку для sync — создать аналогичную функцию discover_*, НЕ хардкодить список.
